## Text Animation Components

- **SplitText** - Character/word reveal animations
- **WaveText** - Hover wave effect on text
- **ScrambleText** - Matrix-style text reveal
- **Typewriter** - Cycling typewriter with cursor
- **RotatingWords** - 3D rotating word carousel
- **MagneticText** - Mouse-following magnetic effect
